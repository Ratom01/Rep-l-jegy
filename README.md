# Repülőjegy
A projekt az adatbázis alapú rendszerek kurzuson készült csapatmunkában. Az adatbázisséma tervezésében és annak adatokkal feltöltésében, egyszerű táblákhoz adatfelvitel lehetősége ürlapon keresztül és ezek lekérdezésében vettem főleg részt, illetve néhány összefüggő lekérdezés megvalósításában.
